Privacy Policy

Last updated: June 16, 2026

This privacy policy explains how the True Education Manhwa Wiki Assistant uses data when interacting with the True Education Manhwa Wiki.

What this GPT does

The True Education Manhwa Wiki Assistant helps users search, read, summarize, and draft wiki content related to the True Education manhwa.

The GPT may use an action connected to the True Education Manhwa Wiki API:

https://true-education-manhwa-wiki-6yn.pages.dev

Information the action receives

When a user asks a question, the GPT may send search terms or page paths to the wiki API.

Examples include:

- Character names
- Chapter numbers
- Organization names
- Wiki page paths
- Search queries related to the wiki

Information the action returns

The wiki API may return:

- Wiki page titles
- Wiki page paths
- Wiki page content
- Search results
- Page excerpts

Personal information

This wiki API is designed for public wiki content. It does not require users to create an account, log in, or provide personal information.

Users should not submit private personal information, passwords, API keys, addresses, payment information, or other sensitive information when using this GPT.

Data storage

The wiki API does not intentionally store user conversations or personal information.

The wiki content is stored in the public GitHub repository connected to the True Education Manhwa Wiki.

Third-party services

This project may use:

- ChatGPT by OpenAI
- GitHub for storing wiki files
- Cloudflare Pages for hosting the website and API

These services may process data according to their own terms and privacy policies.

Contact

For questions about this wiki or privacy policy, contact the site owner through the GitHub repository for the True Education Manhwa Wiki.
